# Air Pollution Monitoring System

## Overview
This project is an IoT-based Air Pollution Monitoring System that measures air quality using sensors and displays real-time data on a mobile dashboard using Blynk.

## Features
- Real-time air quality monitoring
- IoT-based system using ESP32
- Mobile monitoring using Blynk
- Sensor-based pollution detection

## Technologies Used
- ESP32
- IoT
- Blynk
- Gas Sensors

## Components Required
- ESP32
- MQ135 Gas Sensor
- Jumper Wires
- Breadboard
- Power Supply

## Working
1. The sensor detects air pollution levels.
2. ESP32 processes the sensor data.
3. Data is sent to the Blynk cloud.
4. Users can monitor pollution levels on the Blynk mobile app.

## Applications
- Smart Cities
- Environmental Monitoring
- Air Quality Analysis
- Pollution Detection Systems
